Interaction

discussions@password-hashing.net is a public mailing list for discussions regarding the Password Hashing Competition. To subscribe send an empty message to discussions-subscribe@password-hashing.net. This list is unmoderated, and only members are allowed to post messages. Public archives of this mailing list are available thanks to Gmane.

submissions@password-hashing.net should only be used to send submissions to the Password Hashing Competition, as per the call for submissions.

panel@password-hashing.net forwards to the panel of experts; use this address to send any non-public inquiry to the organizers of the competition.

#phc is the official IRC channel of the Password Hashing Competition, on the freenode IRC network. To connect, start an IRC client and type
/connect irc.freenode.net
/join #phc

#pwdhc is the Twitter hashtag.
